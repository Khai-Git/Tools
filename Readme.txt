=======================================================
Cám ơn các bạn đã tin dùng sản phẩm của anhdvshop.com
=======================================================

Giải nén các file ở mục Tool với mật khẩu mặc định: anhdv

1. MAS AIO Active: dùng để kích hoạt Windows hoặc Office

- Để kích hoạt bản quyền số cho Windows 10, 11 Pro thì chọn 1 - HWID
- Để kích hoạt bản đến 2038 năm cho Windows LTSC chọn 2 - KMS38
- Để kích hoạt KMS Online cho Office, Windows 7 hay 8 Pro thì chọn 3 - Online KMS
- Gỡ KMS cho Office, Windows: Chọn 3 - Online KMS => U - Uninstall Completely

2. Windows_Update_Blocker: dùng để chặn Windows Update

Mặc định thì Windows by anhdv đã chặn Auto Update.
Tức là chỉ khi nào vào Windows Settings nhấp vào Check Update thì mới Update
Mình khuyên không cần chặn Update làm gì cả.

3. Defender_Control: dùng để tắt bật Windows Defender

4. Enable_GameBar: dùng để bật GameBar nếu chơi game gặp thông báo yêu cầu bật
